Welcome to What's The Weather!

The Weather page will show you your current location's weather in Farenheit.
To search a different location's weather, please enter the City, State Code, and Country Code for the location.
The State Code is only applicable to US locations, however you can add the state code for other countries if they exist, but both the City name and Country Code are required for a proper search.
You can change the units of measure from Farenheit to Celsius and even Kelvin, which also changes the additional weather information units from imperial to metric, or vice versa.
For users residing in the States, please enter your State Code, such as TX (Texas), NY (New York), CA (California), etc.
*Alternatively, users can also input their Zip or Postal Code, followed by their Country Code.
The weather card displays the current location, the current weather (in selected units), the broad label for the weather, a descriptive label for the weather, a header image that accurately visualizes the current weather type, and a side panel that shows additional information regarding the weather.
The last 24 location/weather seaches are shown in the History tab, press View Details to view the weather again, this time in your default selected units!

The site uses OpenWeathers Weather API and Geocoding API to fetch accurate weather data for the specified location, while it uses the HTML built-in Geolocation API to fetch the current location of the user. Results are dynamically displayed using JavaScript DOM manipulation, combined with the sleek styling capabilities of the Bootstrap CSS library, thus ensuring an organized and visually appealing user experience.

The purpose of this website is to learn how APIs, Bootstrap, and front-end JavaScript works, and to seemlessly implement it together. This project is not monitzed and is strictly built for personal practice purposes.
